## Introduction

The Laravel Calculator is a web application that provides basic calculator functionality. It allows users to perform arithmetic operations like addition, subtraction, multiplication, and division.

## Features

-   Addition of numbers
-   Subtraction of numbers
-   Multiplication of numbers
-   Division of numbers

## Installation

To run the Laravel Calculator locally, follow these steps:

1. Clone the repository:

    ```
    git clone https://github.com/omarharchich/laravel-vuejs-calculator.git
    ```

2. Navigate to the project directory:

    ```console
    cd laravel-vuejs-calculator
    ```

3. Install the dependencies:

    ```
    composer install
    npm install
    ```

4. Generate the application key:

    ```
    php artisan key:generate
    ```

5. Compile the frontend assets:

    ```
    npm run serve
    ```

6. Start the development server:

    ```
    php artisan serve
    ```

7. The Laravel Calculator is now accessible at [http://localhost:8000/calculator](http://localhost:8000/calculator).

## Usage

To use the Laravel Calculator:

1. Open your web browser.
2. Navigate to [http://localhost:8000/calculator](http://localhost:8000/calculator).
3. Perform the desired arithmetic operations by clicking on the number buttons and operation buttons displayed on the calculator interface.
4. View the calculated result on the calculator screen.
